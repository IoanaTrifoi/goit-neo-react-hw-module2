# React Homework 02 - Sip Happens Café Feedback Widget

This project is a feedback collection application for the "Sip Happens Café," developed as part of the **GoIT Neo React Course (Module 2)**. The application tracks customer reviews and persists data across page reloads.

## 🚀 Live Demo
Check out the live application here:
[**Live Page on Vercel**](https://goit-neo-react-hw-module2-mu-one.vercel.app/)

## 🛠️ Technologies Used
* **React** (Functional Components & Hooks)
* **Vite** (Build Tool)
* **CSS Modules** (Scoped Styling)
* **LocalStorage** (Data Persistence)
